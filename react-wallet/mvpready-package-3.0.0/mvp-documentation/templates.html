Notifications

Below is an example of a navbar notification. Use this template to get started creating your own notification items within the navbar.


Example HTML

<li class="dropdown navbar-notification">

  <a href="./page-notifications.html" class="dropdown-toggle" data-toggle="dropdown">
    <i class="fa fa-bell navbar-notification-icon"></i>
    <span class="visible-xs-inline"> Notifications</span>
    <b class="badge badge-primary">3</b>
  </a>

  <div class="dropdown-menu">

    <div class="dropdown-header"> Notifications</div>

    <div class="notification-list">
      <a href="./page-notifications.html" class="notification">
        <span class="notification-icon"><i class="fa fa-cloud-upload text-primary"></i></span>
        <span class="notification-title">Notification Title</span>
        <span class="notification-description">Praesent dictum nisl non est sagittis luctus.</span>
        <span class="notification-time">20 minutes ago</span>
      </a>
    </div>

    <a href="./page-notifications.html" class="notification-link">View All Notifications</a>

  </div>

</li>

</li>

Social Feed

Feed Item Example

An example of the .feed-item html shown below.

<div class="feed-item">

	<div class="feed-icon">
		<i class="icon-lightbulb"></i>
	</div> <!-- /.feed-icon -->

	<div class="feed-subject">
		<a href="javascript:;">Rod Howard</a> shared an idea: <a href="javascript:;">New Idea</a>
	</div> <!-- /.feed-subject -->

	<div class="feed-content">
		<p>
			<i class="icon-quote-left"></i> 
			Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi.
		</p>
	</div> <!-- /.feed-content -->

	<div class="feed-actions">
		<a href="javascript:;" class="pull-left"><i class="icon-thumbs-up"></i> 123</a> 
		<a href="javascript:;" class="pull-left"><i class="icon-comment-alt"></i> 29</a>

		<a href="javascript:;" class="pull-right"><i class="icon-time"></i> 2 days ago</a>
	</div> <!-- /.feed-actions -->

</div> <!-- /.feed-item -->


Feed Item Types

Use feed item types of change the color of the .feed-icon for your post.

Type Class Name
Idea .feed-item-idea
Image .feed-item-image
File .feed-item-file
Bookmark .feed-item-bookmark
Question .feed-item-question

Landing Template

Masthead


Slide Items

The easist method to update the look and feel of a masthead item is to add inline background properties directly to the .item element. An example is below where the background-color and background-image properties are set.

<div class="item active" style="background-color: #354b5e; background-image: url();">

  <div class="container">

    <div class="row">

      <div class="col-md-6 masthead-text">
        <div class="masthead-actions"></div> <!-- /.masthead-actions -->
      </div> <!-- /.masthead-text -->

      <div class="col-md-6 masthead-img">
        <img src="img/masthead/mobile.png" alt="slide2" class="img-responsive" />     
      </div> <!-- /.masthead-img -->

    </div> <!-- /.row -->

  </div> <!-- /.container -->

</div> <!-- /.item -->


Layouts

The layouts of the masthead items are controlled using the Bootstrap grid system. You can create many different layouts using the system along with shifting the position of columns via CSS.
